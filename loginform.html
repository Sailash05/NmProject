Privacy & Security Note

This is a demo HTML-only page. Do not enter real Aadhaar/VID on untrusted pages. In production, always use HTTPS and server-side validation. OTP will be sent to the mobile number registered with Aadhaar.

How it works (typical flow)

  1. User enters Aadhaar number or VID and the captcha.
  2. Server validates input and calls UIDAI authentication API (AUA/ASA integration).
  3. UIDAI sends OTP to the registered mobile number.
  4. User enters OTP on a separate page to complete login.
